Detailed Description

Displayes 1D-CA with images for all states.

While DisplayImages2D and DisplaySquares2D just over-blit the whole display, DisplayImages1D (and, btw, DisplaySquares1D, too) shifts the displayed configuration of earlier steps up and displayes the new configuration in the bottom row (see DisplayImages1D.newlineSurface), creating a space-time-diagram, the downwards directed Y-Axis representing time.
