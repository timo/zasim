:mod:`zasim.gui.main` - Running the gui from the commandline
============================================================

Running the gui from the commandline
------------------------------------

Invoke this module directly, like this

.. command-output:: python -m zasim.gui.main --help
    :shell:

or using the launcher `zasim_gui` from the commandline:

.. command-output:: zasim_gui --help
    :shell:
    :ellipsis: 5
