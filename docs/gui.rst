:mod:`zasim.gui` - Qt based display and user interface
======================================================

Zasim offers a gui frontend for displaying and interacting with cellular
automatons, as well as creating new ones.

The gui package is divided into multiple modules that cover different
distinct parts of the user interface. Each has its own section in this
documentation.
